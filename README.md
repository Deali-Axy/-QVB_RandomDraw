#QVB_RandomDraw
（QVB实现 简单随机抽签器）

### 项目简介：
这是一个使用QVB库开发的例子，简单运用了一些QVB的功能，短短几行代码就实现了一个基本的抽签器，QVB库封装的功能着实方便了程序的开发

### QVB简介：
QVB是我在2013年开发的一个API封装库，封装了一些常用的简单功能，可以方便VB程序的开发

### 开发工具
Visual Basic 6.0
